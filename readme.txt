Практика написанная мною для МУИВ, генерирует кубы, выбирает случайный материал, цвет и местоположение объекта, а потом собирает эти данные в .txt файл в папке Assets

Управление в режиме свободной камеры:
wasd - перемещение
shift - ускорение
space (пробел) - поднять камеру
left ctrl - опустить камеру

Данные программа собирает автоматически после генерации в .txt файл под названием "GameInfo", сам файл находится в папке Assets
